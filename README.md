# 📜 The Constitution Test

The USCIS naturalization exam tests whether you can name the branches of government. **The Constitution Test** asks whether you actually know the document that created them — every Article, every clause, every Amendment, word for word from the enrolled text at [Congress.gov](https://constitution.congress.gov).

This repository contains two tests built on the same engine. Start with the Citizen Test to verify your civics foundation, then take the Constitution Test to prove you know the supreme law of the land.

## ⚖️ About The Constitution Test

| | The Citizen Test | The Constitution Test |
|--|--|--|
| **Source material** | 2025 USCIS 128-question bank | U.S. Constitution — Congress.gov enrolled text |
| **Question bank** | 120 questions | 75 questions |
| **Questions per session** | 55 (randomly selected) | 40 (randomly selected) |
| **Pass threshold** | 50 correct | 34 correct |
| **Fail threshold** | 5 incorrect | 5 incorrect |
| **Answer order** | Randomized every question | Randomized every question |
| **Stops early?** | Yes | Yes |
| **Source** | [USCIS.gov](https://www.uscis.gov/sites/default/files/document/questions-and-answers/2025-Civics-Test-128-Questions-and-Answers.pdf) | [Congress.gov](https://constitution.congress.gov) |

The Constitution Test covers:

- **Article I** — The legislative branch: elections, terms, qualifications, impeachment, the veto, the necessary and proper clause, habeas corpus
- **Article II** — The executive branch: the presidency, the oath, Commander in Chief, treaty ratification, high crimes
- **Article III** — The judicial branch: tenure, jurisdiction, the jury trial requirement, treason
- **Articles IV–VII** — States' relations, the amendment process, the Supremacy Clause, the prohibition on religious tests
- **Amendments I–X** — The Bill of Rights, every clause
- **Amendments XI–XXVII** — Every subsequent amendment through Congressional pay

## ✅ Accuracy & Sources

All questions are derived directly from the enrolled U.S. Constitution as published by Congress at [constitution.congress.gov](https://constitution.congress.gov). Correct answers are cryptographically encoded in the quiz — they are not readable in the page source or browser developer tools.
